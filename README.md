# hongmmunity
hongmmunity / Java SpringBoot, MySql, JPA, intelliJ

# 주제

화홍고 학생들을 위한 Community 성격의 웹 페이지 제작

# 기능

- 회원가입, 로그인
- 경기도 코로나 현황 확인 기능
- 학생 커뮤니티 게시판
